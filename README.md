### 🔧 Amar Repair — Multi-Tenant SaaS
End-to-end repair business management with strict tenant data isolation and RBAC.

- Built **secure REST APIs** with JWT authentication and role-based access control

Android app for tutors to manage students, attendance, and fees — fully functional without internet.

- Designed efficient local schema optimized for fast queries
- Reduced manual administrative workload by **~80%**
